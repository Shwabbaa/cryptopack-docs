Armour
***************

Hats
==========

Hats are remodelled pumpkin heads. The pumpkin overlay is removed from this resourcepack.
Hats do not give any armor points, they are only for cosmetic value

Baphomet Mask:

* Pattern: "baphomet"
* Items: Carved Pumpkin
* ID: "baphomet_mask"

Meteor Hat:

* Pattern: "meteor"
* Items: Carved Pumpkin
* ID: "meteor_hat"
